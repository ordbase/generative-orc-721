New to BTC Orange Pixels? See the BTC Orange Pixels Getting Started Guide and Documentation »

Help - Left click puts a pixel down, right click clears it.
↓ Print - Generates the g numbers in the textbox to copy-n-paste.
↑ Read - Reads the g numbers in the textbox to draw pixels in canvas.
⊘ Reset - Clears the canvas - resets to empty.
Try Random (Design) - Draws ready-made pixels design.
